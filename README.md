## About
GCS TERMINAL GAME is a collection of **25 mini terminal-based games** that you can play directly in your terminal or Termux. Lightweight, fun, and easy to run! Perfect for anyone who loves quick gaming sessions in a CLI environment.  

---

## Features
- 25 mini games in one repo
- Lightweight & fast
- Easy to install and run
- Compatible with Termux & Linux
- Fun and engaging for all ages
